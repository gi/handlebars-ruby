A simple interface to [Handlebars.js](https://handlebarsjs.com) for Ruby.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'handlebars-engine'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install handlebars-engine
